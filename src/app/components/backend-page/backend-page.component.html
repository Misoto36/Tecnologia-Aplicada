DESARROLLO BACKEND

El desarrollo del backend es la parte del desarrollo que se encarga de manejar la lógica detrás de la interfaz visible y que garantiza el correcto funcionamiento de la aplicación.

Lenguajes Aplicados

C# - .Net

C# es un lenguaje de programación orientado a objetos, multiparadigma y de alto nivel. Es un lenguaje de programación de propósito general, desarrollado por Microsoft. C# es el lenguaje de programación más utilizado en la actualidad, ya que se utiliza en aplicaciones de escritorio, web, servidores, etc.

Se caracteriza por ser un lenguaje fuertemente tipado, permitir realizar operaciones asincronas y facilitar en manejo de estructuras complejas. Actualmente se utiliza en la plataforma .Net y con una amplia cantidad de paquetes y librerias para generar codigo mas estructurado.

Javascript - NodeJs

Javascript es un leguaje de programacion orientado a objetos, su propósito principal esta del lado del Fronted mas sin embargo tambien se puede implementar del lado del servidor utilizando un entorno de ejecucion basado en este lenguaje conocido como NodeJs, el cual nos permite poder realizar tareas de manera asincronas y manejar estructuras complejas, realizar conexiones con la base de datos y ejecutar distintas peticiones a la misma

PHP

PHP es un lenguaje de programación de propósito general, desarrollado por la empresa estadunidense de software británica PHP: Hypertext Preprocessor. Es ampliamente utilizado del lado del servidor para el desarrollo de aplicaciones y paginas web, se caracteriza por poder incrustar su codigo dentro del Html siendo ejecutables sus scripts del lado del cliente aun asi manteniendo su codigo subyasente oculto para que pueda ser visto por el.

Actualmente se utiliza en conjunto con Wordpress un sistema de gestion de contenidos y ya conocido como framework por defecto para PHP, permitiendo crear rapidamente paginas web dinamicas.
