Javascript

Javascript es un lenguaje de programación interpretado, orientado a objetos, esecial para el desarrollo qeb permitiendo agregar dinamismo , interactividada y funciones complejas que permitan brindar una experiancia de usuario muy agradable.

Características del lenguaje Javascript

Javascript es un lenguaje débilmente tipado dado que no es necesario definir un tipo de variable cuando esta es declarada, así mismo permite ser dinámico dado que en tiempos de ejecución las variables pueden ser reasignadas con un tipo de dato distinto.

Es un leguaje muy intuitivo, posee una estructura sencilla y fácil de entender por lo cual también es un leguaje ideal para aquellos que apenas están empezando en el área de la programación y el desarrollo web, además es un lenguaje que no necesita compilador por lo cual puedes ejecutarlo directamente desde el navegador.

Recuerda que la mayoría de las bases en programación son generales y aplica para todos los leguajes, es importante familiarizarte con la sintaxis del mismo y adentrarse en los fundamentos de la programación, saber identificar los algoritmos, conocer las principales sentencias de control, estructuras de datos, creación de funciones y en este caso particular para el desarrollo web algo fundamental la manipulación del DOM.

Document Object Model DOM

El Document Object Model (DOM) es un modelo de objetos que representa el contenido y estructura de una página web.

El DOM se interpreta como un árbol y cada rama se desprenden nodos, cada nodo pude hacer referencia a un elemento dentro de nuestro documento y utilizando JavaScript podemos acceder a ellos manipularlos modificar sus clases y atributos, o por otra parte podemos crear nuevos elementos, adecuarlos a las necesidades para generar esa interactividad de la que el usuario final va hacer partícipe.
