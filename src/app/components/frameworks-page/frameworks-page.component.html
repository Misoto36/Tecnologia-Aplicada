FRAMEWORKS

Los frameworks son tecnologías diseñadas que brindas bases para la creación de proyectos de carácter o con objetivos específicos, su propósito es facilitar el desarrollo brindando simplicidad ayudando a optimizar tiempos de trabajo, promoviendo el trabajo colaborativo, mejorando la arquitectura de los proyectos para obtener resultados de calidad. Podemos encontrar frameworks para todo tipo de desarrollo dependiendo su propósito y el lenguaje al que este orientado. En esta oportunidad hablaremos de uno especialmente utilizado para el desarrollo web con alta demanda por el sector empresarial y sistemas financieros.

Angular es un framework para desarrollo web que permite la creación eficiente de aplicaciones escalables, cuenta con gran cantidad de librerías y herramientas disponibles para que los desarrolladores puedan optimizar su flujo de trabajo.

Siendo un framework basado en componentes permite no solo mejorar la calidad de nuestro código si no también permite la optimización de recurso con la reutilización de elementos. En este articulo no vamos profundizar sobre el framework solo algunos aspectos importantes, si deseas aprender y adentrarte en él, puedes visitar su documentación oficial con su versión más reciente aquí.

Información adicional

Angular está construido con TypeScript y su documentación y herramientas están orientadas a TypeScript. Esto significa que obtienes una experiencia más coherente y optimizada cuando usas TypeScript con Angular. Contexto: TypeScript es un lenguaje derivado de JavaScript proporciona un sistema de tipos estáticos que ayuda a detectar errores en tiempo de compilación, antes de que el código se ejecute en el navegador, y puedes obtener un desarrollo mas robusto, eficiente y seguro.

LaReactividades un concepto con el que se trabajara mucho en Angular, y basicamente hace referencia a la capacidad que tiene el framwork para detectar cambios de estado y datos y poder brindar una respuesta de manera eficiente a los mismos. Se puede identificar en lo que dentro del framework se conoce como formularios reactivos, o tambien al realizar peticiones mediante servicios al servidor. Al hablar de reactiviadad una de las librerias mas populares es RxJs.

Angular Material es una de las librerias con las que cuenta Angular, permite poder obtener componentes ya construidos para facilitar y optimizar el desarrollo de aplicaciones, entre estos encuentras tipos de input, tablas, paginadores, cards entre otros, en su documentacion se puede ver como incorporarlos al proyecto con instructivos faciles de comprender.
