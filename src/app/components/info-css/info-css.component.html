CSS

CSS es un lenguaje de estilos en cascada que se utiliza para definir la presentación de los documentos HTML. Su primera versión apareció en 1996 y desde entonces a evolucionado a versiones CSS 1, CSS 2 y la más actualizada y aun en estado de evolución CSS 3. El termino cascada básicamente nos da a entender que es importante el orden en que aplicamos los estilos. Se pueden aplicar directamente estilos desde nuestro archivo html utilizando la etiqueta <stylel> dentro del <head> tambien ubicando los estilos en línea dentro elemento mediante el atributo style="" ubicado en la etiqueta de apertura del elemento y como en un archivo independiente de extensión .css para brindar mejor estructura en la arquitectura de nuestro proyecto.

Anatomía de un selector CSS

h1 {{ "{" }}
color: red;
{{ "}" }}

h1 es el selector, color:red; es la propiedad y red; es el valor de la propiedad.



Tipos de selectores en CSS

El selector en css básicamente nos ayuda a identificar el elemento al cual deseamos aplicar el estilos.

Selector universal * : Aplica o define estilo para ciertas propiedades generales en todo nuestra página.

Selector de etiqueta : Aplica o define estilo para ciertas etiquetas en toda nuestra página.

Selector de clase : Aplica o define estilo para ciertas clases en todo nuestra pagina.La clase se define en la etiqueta de apertura html con el atributo class="". <h1 class="titulo">, el nombre de la clase pude utilizarse en varios elementos html por lo cual facilita aplicar el mismo estilo a distindos elementos sin tener que escribir muchas lineas dentro de nuetro css para cada uno por separado.

Selector de id : Aplica o define estilo para un id en nuestra pagina.El id se define en la etiqueta de apertura html con el atributo id="". <h1 class="titulo" id="principal">, Cabe aclarar que el id es unico y solo se aplica a un solo elemento html.

Selector de atributo : Aplica o define estilo para ciertos atributos en todo nuestra página. a[href="#"] selecciona todos los enlaces con el atributo href igual a #.



Estilos que puedes aplicar

CSS cuenta con una gran variedad de propiedades que puedes aplicar para que la página o aplicación sea atractiva para el usuario o cliente final, puede cambiar los tipos de fuentes de tu letra, los tamaños de contenedores y sus márgenes, colores, animaciones, realmente todo cabe en la imaginación, puedes obtener la guía completa de estilos accediendo a la documentación oficial.



Ubicación de los Elementos

Un aspecto importante en el diseño de nuestra página o aplicación web es la forma en que presentamos nuestros elementos, su ubicación en el espacio de la página para dar una presentación armoniosa para lo cual explicaremos 3 conceptos claves.



Box Model

En CSS, todo elemento se representa como una caja rectangular. Esta caja tiene cuatro partes principales:

El contenido:El contenido del elemento pude ser texto, imagenes etc.

El borde:Es la linea que rodea el contenido y relleno del contenedor.

El padding:Es el espacio entre el contenido y el borde.

El margen: la distancia el elemento y elementos cercanos.

Flex box

El flexbox es un sistema de posicionamiento que permite alinear y distribuir los elementos en una fila o columna.

flex-direction: Define la direccion en la que se alinean los elementos.

justify-content: Define la alineacion horizontal de los elementos.

align-items: Define la alineacion vertical de los elementos.

align-content: Define la alineacion vertical de los elementos.

CSS Grid

El grid es un sistema de posicionamiento que permite alinear y distribuir los elementos en una fila o columna.

grid-template-columns: Define la cantidad de columnas.

grid-template-rows: Define la cantidad de filas.

grid-template-areas: Define la ubicacion de los elementos.

grid-gap: Define el espacio entre los elementos.
