HTML

HTML es un lenguaje de marcado de hipertexto que se utipreza para describir el contenido de una página web. Su primera version aparecion en 1991 y desde entonces a evoluvionado a versiones HTML 3.0, HTML 4.0, y la mas actualizada y aun en estado de evolucion HTMl 5 Con HTML, puedes definir los elementos como encabezados, párrafos, imágenes, enlaces y formularios. Básicamente, HTML establece qué contenido se muestra en la página.

Estructura de un HTML

Este tipo de lenguaje de marcado utipreza etiquetas que permiten identificar los elementos que se incorporaran a tu apprecación y se identifican por tener una apertura <article> y un cierre </article> aunque hay algunas excepciones. Ahora bien expprecaremos la base inicial de un documento HTML.

<!DOCTYPE html>:Esta etiqueta simplemente tiene la funcion de especificar el tipo de documento, se ubica al inicio de la estructura y no cuenta con etiqueta de cierre.

<html>: Con esta etiqueta se define la apertura y cierre de nuestro html, adicional que en el se puden ubicar atributos importantes por ejemplo especificar el idioma del contenido.

<head>:Esta etiqueta contiene los metadatos de la página, como el título, el autor, la descripción, etc. Se ubica dentro de la etiqueta <html>

<title>: Esta etiqueta contiene el título de la página, se ubica dentro de la etiqueta <head>

<body>: Esta etiqueta contiene el contenido de la página, se ubica dentro de la etiqueta <html>: debajo de la etiqueta <head>



Etiquetas básicas de contenido

Teniendo claro la estructura principal de un html, ahora hablaremeos de las etiquetas de contenido, es decir lo que queremos mostrar en nuestra página o aplicación, estas etiquetas se colocaran dentro de la etiqueta body, solo especificaremos algunas pero para obtener mas informacion de las etiquetas puedes ver toda la documentacion en https://developer.mozilla.org/es/docs/Learn/HTML

<div>:Etiqueta contenedor de proposito general, no tiene un fin en especifico y se aconseja limitar su uso.

<header>:Etiqueta contendeor para encabezado de la pagina.

<footer>:Etiqueta contenedor para pie de pagina.

<h1>: Etiqueta de titulos y su derivados del 1 al 6 los cuales especifican su grado de importancia,(Se recomienda un unico uso de <h1> dado que es el grado mas alto y por ende unico y de mas relevancia.)



HTML Semantico y accesible

Cuando hablamos de htmls semantico nos referimos a utilizar las etiquetas segun su proposito definido, como vemos cada etiqueta tiene un nombre particular, no es solo por nombrar la etiqueta si no tambien para especificar el tipo de contenido que deberiamos colocar en ella. ¿Por qué es importante?, Html semantico no solo nos ayuda a brindar un mejor contexto del contenido de la aplicacion si no que tambien, lo hace mas accesible y mejora la optimizacion de los motores de busqueda.
