HTML

HTML es un lenguaje de marcado de hipertexto que se utiliza para describir el contenido de una página web. Su primera versión aparición en 1991 y desde entonces a evolucionado a versiones HTML 3.0, HTML 4.0, y la mas actualizada y aun en estado de evolución HTMl 5 Con HTML, puedes definir los elementos como encabezados, párrafos, imágenes, enlaces y formularios. Básicamente, HTML establece qué contenido se muestra en la página.

Estructura de un HTML

Este tipo de lenguaje de marcado utiliza etiquetas que permiten identificar los elementos que se incorporaran a tu aplicación y se identifican por tener una apertura <article> y un cierre </article> aunque hay algunas excepciones. Ahora bien explicaremos la base inicial de un documento HTML.

<!DOCTYPE html>:Esta etiqueta simplemente tiene la función de especificar el tipo de documento, se ubica al inicio de la estructura y no cuenta con etiqueta de cierre.

<html>: Con esta etiqueta se define la apertura y cierre de nuestro html, adicional que en él se pueden ubicar atributos importantes por ejemplo especificar el idioma del contenido.

<head>:Esta etiqueta contiene los metadatos de la página, como el título, el autor, la descripción, etc. Se ubica dentro de la etiqueta <html>

<title>: Esta etiqueta contiene el título de la página, se ubica dentro de la etiqueta <head>

<body>: Esta etiqueta contiene el contenido de la página, se ubica dentro de la etiqueta <html>: debajo de la etiqueta <head>



Etiquetas básicas de contenido

Teniendo claro la estructura principal de un html, ahora habláremos de las etiquetas de contenido, es decir lo que queremos mostrar en nuestra página o aplicación, estas etiquetas se colocarán dentro de la etiqueta body, solo especificaremos algunas, pero para obtener más información de las etiquetas puedes ver toda la documentación. aquí.

<div>:Etiqueta contenedor de proposito general, no tiene un fin en específico y se aconseja limitar su uso.

<header>:Etiqueta contendeor para encabezado de la página.

<footer>:Etiqueta contenedor para pie de página.

<h1>: Etiqueta de titulos y su derivados del 1 al 6 los cuales especifican su grado de importancia, (Se recomienda un unico uso de <h1> dado que es el grado más alto y por ende único y de más relevancia.)



HTML Semántico y accesible

Cuando hablamos de html semántico nos referimos a utilizar las etiquetas según su propósito definido, como vemos cada etiqueta tiene un nombre particular, no es solo por nombrar la etiqueta si no también para especificar el tipo de contenido que deberíamos colocar en ella. ¿Por qué es importante?, Html semántico no solo nos ayuda a brindar un mejor contexto del contenido de la aplicación si no que también, lo hace más accesible y mejora la optimización de los motores de búsqueda.
